# Установка 3X-UI-pro with REALITY

Скрипт для быстрой установки панели управления 3X-UI с поддержкой протокола REALITY на порту 443.

## Команда для установки

```bash
sudo su -c "bash <(wget -qO- https://raw.githubusercontent.com/mozaroc/x-ui-pro/refs/heads/master/x-ui-pro.sh) -install yes -auto_domain y"
```

## Особенности
- Автоматическая настройка домена.
- Оптимизация ядра под высокую нагрузку.
- Поддержка XTLS-Vision и Reality «из коробки».
