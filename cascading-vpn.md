# Каскадный VPN (Chain: Client -> Russia -> World)

Инструкция по настройке цепочки серверов для обхода сложных блокировок.

## Схема работы
1. **Клиент (Роутер/Смартфон)** подключается к серверу в РФ.
2. **РФ Сервер** перенаправляет трафик на зарубежный сервер (Финляндия/Нидерланды).
3. **Зарубежный сервер** выпускает трафик в свободный интернет.

## Настройка на стороне РФ сервера (3X-UI)
1. Создайте **Inbound** (VLESS + Reality) для приема входящего трафика.
2. Создайте **Outbound** (VLESS + Reality) с данными вашего зарубежного сервера.
3. В разделе **Routing** создайте правило:
   - `Inbound Tag`: имя вашего входящего подключения.
   - `Outbound Tag`: имя вашего зарубежного подключения.

Это создаст прозрачный мост между серверами.
